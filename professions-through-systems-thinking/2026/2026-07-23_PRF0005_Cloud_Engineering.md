## Overview

Cloud Engineering is more than using cloud platforms.

It is the discipline of designing, building, deploying, operating, and continuously optimizing cloud systems that are secure, scalable, reliable, and cost-effective.

This public learning board introduces Cloud Engineering from a systems thinking perspective, emphasizing relationships between infrastructure, applications, services, automation, security, observability, governance, and continuous improvement.

---

## Learning Topics

1. What is Cloud Engineering?
2. Why Cloud?
3. Systems Thinking for Cloud Engineering
4. Cloud Engineering Life Cycle
5. High-Level Cloud Architecture
6. Good Cloud Systems
7. Common Trade-offs
8. Key Principles
9. Cloud Engineer Mindset
