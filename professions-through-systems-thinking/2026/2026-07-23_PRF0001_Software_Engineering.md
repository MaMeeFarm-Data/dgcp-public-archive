## Overview

Software Engineering is more than writing code.

It is the discipline of designing, building, operating, and continuously improving systems that create long-term value.

This public learning board introduces Software Engineering from a systems thinking perspective, emphasizing relationships between people, processes, technology, architecture, operations, and continuous improvement.

---

## Learning Topics

1. What is Software Engineering?
2. Software vs System
3. Systems Thinking
4. Software Development Life Cycle
5. System Architecture
6. Good Systems
7. Common Trade-offs
8. Key Principles
9. Engineer Mindset
